Proyecto académico enfocado en el desarrollo de una plataforma web para incentivar el reciclaje mediante un sistema de recompensas.

Participé activamente en el diseño de la base de datos, elaborando el modelo entidad-relación, así como en la implementación de triggers y procedimientos almacenados en SQL Server.

Tecnologías utilizadas: Node.js, SQL Server, React.
